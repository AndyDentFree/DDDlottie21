Integrating a Lottie Animation

Flow Version 1.11.1

This document explains how to run the iOS Xcode project you have just exported using Flow’s Lottie-iOS export.

When working through this doc, remember:

  • The code below contains variables from your Flow file.
  • That means you can copy and paste it verbatim into your project.
  • We used our code export engine to produce it, so you can be sure that it is exact

Custom Code

Every time you export a timeline to code it varies depending on the animations, layer names, etc. To ensure these instructions are crystal clear they are tailored to the code you have just exported. For example, the name of the timeline you just exported is timeline.

Let’s get to it.

Folder Breakdown

When you export a project you’ll see the following structure:

  • Timeline - Folder containing everything for your animation and demo Xcode project.
    • img - Folder containing all the image files
    • Resources - Folder containing important assets including the Lottie Animation file.
    • Resources/LottieAnimations/timeline.min.json - The actual Lottie JSON file, gloriously minified
    • Sources - Folder containing all the code for your animation apart from the Lottie Animation file
      • FlowCommon - A library containing the code used to play the iOS animations. Optionally, install via the FlowCommon cocoapod.
      • Launch Animation - The launch animation played when you run Timeline.xcodeproj

Running the iOS Xcode Project

Here, we provide some simple steps for running the Lottie - iOS export example Xcode project.

  1. Open the newly generated workspace file: 
                                                Timeline.xcworkspace
  2. Run it!

  3. Once the app is running, you can play the animation by clicking on the artboard.

    4. Note:
    If the Timeline.xcworkspace file was not generated, it is likely because cocoapods is not installed on your device. In this case, install cocoapods on your device by following the instructions provided here and run the following command in terminal: 

                                        cd '/Users/aussiedev/dev/DDDlottie21/animationOutput/DDDlogo/Lottie - iOS/Timeline'; pod install

    Running this command should generate the Timeline.xcworkspace file. You can now follow the instructions provided to run your app.
